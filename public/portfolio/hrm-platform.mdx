Developed a full-featured Human Resource Management (HRM) platform as part of a collaborative team effort, utilizing a modern JavaScript stack with Next.js, Express, Prisma, Tailwind CSS, and Framer Motion, all written in TypeScript.

The project involved both frontend and backend development, ensuring seamless integration and communication between client and server. Building dynamic, responsive features and delivering a refined, user-friendly UI with smooth animations and modern design principles.
