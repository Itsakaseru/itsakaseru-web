# Ex-Send
<InfoBubble
    info={[

    ]}
    color="dayker"

EXSend is a delivery simulation program where both customers and couriers can log in. Customers can create orders and view the final shipping price, while couriers can check customer orders and calculate optimal delivery routes using the Dijkstra Algorithm. This project was developed as part of our Algorithms and Data Structures final project and marks our first programming project in C.

In an unfortunate situation, our team was assigned one of the most difficult project topics available. As the team leader, I guided the group through the challenge, coordinating tasks and ensuring smooth collaboration. Despite the difficulty, we successfully delivered a fully functional simulation program, giving me valuable experience in team management, problem-solving, and implementing algorithmic logic in a real-world project.
