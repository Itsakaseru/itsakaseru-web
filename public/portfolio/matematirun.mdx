MateMatirun is a math-learning rhythm game where players solve math questions while following the rhythm of the song until it ends. Developed as part of our Game Design and Development 1 final project, the game combines ideas from the entire team into a single cohesive experience, we think it turned out spectacular! This project marks our first full game built using Unity after months of learning the game development.

I led the team for this project, taking primary responsibility for the game design, including UI and UX, and collaboratively creating the Game Design Document (GDD) together with the team. I also contributed to the backend development, helping implement core gameplay mechanics and ensuring smooth integration between design and functionality.
