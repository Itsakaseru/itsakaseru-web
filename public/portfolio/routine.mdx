# Routine

Routine is a simple reminder and digital wellbeing application developed using the Android SDK as part of our Mobile Application Programming final project. The application functions as a smart alarm clock that automatically repeats based on the user’s daily routine.

In addition to reminders, the app includes basic digital wellbeing features that allow users to track how consistent they are with their daily routines. The goal of the application is to encourage better habits and help users build consistency in their everyday activities.

For this project, I led the team, overseeing both design and development efforts. I was responsible for the UI/UX design of the application and collaborated closely with the team on both frontend and backend components, ensuring that the design was effectively implemented and fully aligned with the app’s functionality.
