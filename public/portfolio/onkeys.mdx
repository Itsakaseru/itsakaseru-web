onKeys is a rhythm-based typing game built using Unity Game Engine, developed originally for my bachelor thesis project using Unity 2021.3.0f1. I've since made some improvements to the game design and used it as a research project for my master thesis to determine whether a significant effect can be observed in playersâ€™ typing speed and accuracy before and after playing the game. The project has since been upgraded to Unity version 2022.3.20f1 and several game components have been added.

Currently, the game is undergoing a massive redesign from the ground up, named Project Reboot, which will be the next generation of onKeys. Although information regarding the development of this project is kept private, the project reboot community discord server is open to the public. If you wish to join, you can do so via this [link](https://reboot.skyencripttion.com/discord).
