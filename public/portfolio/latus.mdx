# Latus

Latus is a simple social media website developed as part of our Web Programming mid-term project, built using HTML, CSS, and pure PHP. Users can discover "friends", visit their profiles, comment on posts, and interact in ways similar to Twitter, but with a more open, public, and simpler approach. Additional features include updating profile pictures, customizing profile themes, adding background images, and creating posts with text or images.

I led the team for this project, overseeing both design and development efforts. I contributed primarily to the website design (UI/UX) while collaborating with the team on backend development, ensuring smooth integration between frontend and backend components. This project marked my first hands-on experience building a full-stack web application, providing valuable insight into project management, team coordination, and real-world web development.
