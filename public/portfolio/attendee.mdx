A simple modern Attendance Management Platform designed for efficient tracking and management of employee or student attendance. The application is built using React Router v7 (formerly Remix) to enable seamless navigation and robust route management.

This project was developed as part of an interview test and completed within a 7-day timeframe, showcasing full-stack frontend and backend integration. It represents my first attempt at designing a simple microservices architecture and organizing the codebase using a monorepo structure. While the core features are fully implemented and ready for demonstration, there are a few minor frontend and backend error-handling improvements that can be addressed in future iterations.
