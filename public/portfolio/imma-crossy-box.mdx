# Imma Crossy Box

Imma Crossy Box is a simulation game where you play as a box with a single objective: deliver itself into someone’s house. Inspired by the mobile game Crossy Road, we aimed to increase the challenge by reimagining the main character as a box and introducing multiple input controls, creating a movement system that is intentionally simple yet surprisingly demanding.

This game was developed as part of our Computer Graphics and Animation final project. By experimenting with unconventional controls and movement mechanics, we explored how simple systems can create engaging, frustrating, and fun gameplay at the same time.

For this project, I coordinated the team’s efforts and took primary responsibility for the game design, including UI, UX, and the Game Design Document (GDD) which we developed collaboratively. I also contributed to backend development, helping implement core gameplay logic and systems. The final result exceeded my initial expectations, demonstrating how simple mechanics can deliver a fun and engaging player experience.
